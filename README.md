# Excel Project: Interactive Data Analysis Dashboard

## Overview

This project demonstrates the use of Microsoft Excel for data analysis, reporting, and dashboard creation. The workbook transforms raw data into actionable insights through data cleaning, transformation, visualization, and interactive reporting features.

The goal of this project was to create a user-friendly dashboard that enables stakeholders to quickly analyze key metrics and make data-driven decisions.

## Features

- Data cleaning and preparation
- Interactive dashboard design
- Pivot Tables and Pivot Charts
- Dynamic filtering using Slicers
- KPI tracking and performance metrics
- Automated calculations using Excel formulas
- Professional data visualizations
- User-friendly report layout

## Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Validation
- Advanced Excel Formulas
  - IF Statements
  - XLOOKUP / VLOOKUP
  - SUMIFS
  - COUNTIFS
  - INDEX & MATCH
  - TEXT Functions

## Dashboard Components

### Key Performance Indicators (KPIs)
- Total sales/revenue
- Average performance metrics
- Trend analysis
- Category comparisons

### Interactive Filters
Users can filter dashboard results based on:

- Product categories
- Geographic regions
- Time periods
- Customer segments

### Visualizations
- Bar Charts
- Column Charts
- Line Charts
- Pie Charts
- KPI Cards

## Project Workflow

1. Import raw dataset
2. Clean and validate data
3. Create calculated fields
4. Build Pivot Tables
5. Design interactive dashboard
6. Add slicers and filters
7. Format and optimize visualizations
8. Deliver final reporting solution

## Business Value

This dashboard helps users:

- Identify trends and patterns
- Monitor business performance
- Track important KPIs
- Compare categories and segments
- Improve decision-making using data
